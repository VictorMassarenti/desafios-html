Status Code

200
Resposta do servidor informando que está tudo ok.
204
Resposta do servidor informando que está tudo ok, porém o servidor não tem nenhum conteúdo a devolver.
301
O servidor recebeu a mensagem e redirecionou o request para outra página.
404
Página não encontrada.
500
Erro interno do servidor.
